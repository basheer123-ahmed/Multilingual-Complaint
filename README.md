# Multilingual AI-Powered Interactive Complaint Portal (CGMP)

An advanced, full-stack central grievance management portal that leverages artificial intelligence to streamline complaint submission, translation, and resolution. Built for performance, modularity, and scalability, this system offers an intuitive modern UI for Citizens, Officers, and Admins.

## 🚀 Key Features

### 🤖 AI-Powered Capabilities
- **Multilingual Support & Translation:** Automatically detects language, translates user inputs to English, and responds natively.
- **Intelligent Intent Extraction:** Parses user queries and extracts key metrics (e.g., complaint type, severity).
- **Automated Categorization & Prioritization:** AI analyzes complaints and automatically assigns appropriate risk levels.
- **Smart Chatbot Assistant:** Interactive AI assistant capable of answering data-driven questions and guiding users without hallucinating.

### 💻 Core Portal Features
- **Role-Based Access Control (RBAC):** Distinct dashboards and interfaces for Citizens, Officers, and Admins.
- **Interactive Dashboards & Map Views:** Real-time KPI cards, analytics visualizations, and geographical mapping of complaints.
- **Case Management System:** Register cases, update statuses, upload files securely, and manage officer assignments directly.
- **Responsive Premium Design:** "Billion-dollar SaaS" aesthetic using modern UI/UX principles, dynamic animations, and responsive layouts.

## 🛠️ Tech Stack

- **Frontend:** React, Vite, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose ORM)
- **AI Integrations:** Automated Translation, Intent Parsing, Chatbot Engine

## 📦 Setup and Deployment

### 1. Clone the Repository
```bash
git clone https://github.com/Abhinay-12-k/Multilingual-AI-Powered-Interactive-Complaint-Portal.git
cd Multilingual-AI-Powered-Interactive-Complaint-Portal
```

### 2. Backend Setup
Navigate to the backend and configure environment variables.
```bash
cd backend
npm install
npm run dev
```

### 3. Frontend Setup
Navigate to the frontend and configure environment variables.
```bash
cd frontend
npm install
npm run dev
```

Ready for seamless deployment on platforms like Render or Vercel out of the box.
